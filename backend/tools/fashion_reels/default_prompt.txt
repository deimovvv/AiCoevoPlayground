━━━ FASHION REEL RULES (MANDATORY) ━━━

CONCEPT:
- Classic "outfit transition" reel: same model, same spot, different outfits
- Each frame should feel like a still from a fashion campaign
- Transitions will be created by animation (Kling) — you provide the static frames
- Plan 3-6 frames depending on number of outfits available

MODEL CONSISTENCY (CRITICAL):
- The model must look IDENTICAL in every frame
- Same hair style, same makeup level, same skin
- Same body position relative to camera (can vary pose slightly)
- Same camera distance and angle across all frames
- This is essential for smooth outfit-transition animations

POSE DIRECTION:
- Poses should be fashion-forward but not extreme
- Subtle variations between frames: hand position, weight shift, head tilt
- Movement should feel like a fashion film, not a catalog
- Confidence and ease in every pose
- Natural interaction with the garment: touching fabric, adjusting collar, hand in pocket

LIGHTING:
- Consistent lighting across ALL frames — same setup, same direction, same intensity
- Fashion lighting: soft key + rim light for separation
- Studio: clean, controlled, dramatic or minimal
- Location: consistent natural light (same time of day feel)

COMPOSITION:
- Full body or 3/4 shot — SAME framing in every frame
- Centered or consistent off-center placement
- Clean background that doesn't distract from outfit changes
- Vertical 9:16 format optimized for Reels/TikTok

GARMENT SHOWCASE:
- Each outfit must be fully visible and clearly different from the others
- Fabric textures, colors, and silhouettes should be distinguishable
- Styling should be complete (shoes, accessories if relevant)
- The garment is the star — model serves the fashion

STYLE:
- Ultra-realistic, 8K quality
- Fashion campaign photography level
- Natural skin texture, real fabric physics, no AI artifacts
- Color grading consistent across all frames (same edit/look)
- Must feel like frames from a real fashion reel, not AI art

Return ONLY a valid JSON array. No markdown, no explanation.
